# CodeAlpha Bioinformatics Internship Tasks

This repository contains the bioinformatics tasks completed during the CodeAlpha Bioinformatics Internship Program.

## Completed Tasks

### Task 1 — DNA/Protein Sequence Analysis

Performed BLASTP analysis on the human TP53 protein sequence using UniProt and NCBI BLAST to identify homologous protein sequences and analyze evolutionary conservation across species.

### Task 2 — Multiple Sequence Alignment (MSA)

Conducted Multiple Sequence Alignment of TP53 proteins from multiple organisms using Clustal Omega to identify conserved regions, sequence similarities, and evolutionary variations.

### Task 3 — Protein Structure Prediction

Visualized and analyzed the 3D structure of TP53 protein and identified important structural elements including alpha helices, beta sheets, and loop regions involved in protein folding and function.
